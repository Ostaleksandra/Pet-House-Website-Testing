# Pet House Website Testing

## Описание проекта

В рамках данного проекта проведено функциональное тестирование веб-сайта **"Дом питомца"**.

Цель тестирования — проверить корректность работы основных функций сайта и выявить дефекты пользовательского интерфейса.

Тестирование проводилось для страницы:
http://158.160.56.133/app/pets

---

## Объект тестирования

В ходе тестирования проверялись следующие элементы сайта:

* навигация в шапке сайта
* поиск питомцев
* карточки питомцев
* кнопки фильтрации ("Без дома", "Счастливчики")
* форма "Приютить питомца"
* ссылки на социальные сети
* валидация полей формы

---

## Тестовая документация

В проекте представлены следующие артефакты тестирования:

### Чек-лист

Чек-лист содержит проверки основных функций сайта:

* работа кнопок навигации
* корректность поиска питомцев
* отображение карточек животных
* работа фильтров
* проверка формы "Приютить питомца"

---

### Тест-кейсы

Разработаны тест-кейсы для проверки ключевых сценариев работы сайта:

* загрузка страницы "О нас"
* поиск питомца по имени
* работа кнопки "Без дома"
* открытие формы "Приютить питомца"

Каждый тест-кейс содержит:

* приоритет
* предусловия
* шаги выполнения
* ожидаемый результат
* фактический результат

---

### Баг-репорты

В процессе тестирования были выявлены дефекты:

Примеры найденных багов:

* некорректная работа поиска по породе
* отображение на странице животных, не соответствующих требованиям
* некорректная работа кнопки "Питомцы без дома"
* ошибка работы кнопки "Счастливчики"

Все дефекты оформлены в виде баг-репортов.
